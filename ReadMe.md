# Downloader

Internet Download Maneger Written in Go

## Usage

- `git clone git@github.com:ndiecodes/downloader.git`
- `cd downloader`
- `go build`
- `./downloader -u "https://example.com/video.mp4"`

### Flags

- `-u` - Url of the file you want to download
- `-n` - rename downloaded file (optional)
