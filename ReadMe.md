# Downloader

Internet Download Manager Written in Go

## Usage

- `go get github.com/ndiecodes/downloader`
- `downloader -u "https://example.com/video.mp4"`

### Flags

- `-u` - Url of the file you want to download
- `-n` - rename downloaded file (optional)
